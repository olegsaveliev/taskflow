# Screenshots

This folder contains screenshots for the User Guide.

## Required Screenshots

Take these screenshots from the running app (http://localhost:5173):

| Filename | What to Capture |
|----------|-----------------|
| `login.png` | Login page with empty form |
| `signup.png` | Registration page with empty form |
| `dashboard.png` | Dashboard with a few tasks visible |
| `new-task-button.png` | Dashboard highlighting the "+ New Task" button |
| `task-form.png` | Task creation form |
| `task-card.png` | A single task card (highlight the checkbox) |
| `edit-button.png` | Task card with edit (pencil) icon highlighted |
| `delete-button.png` | Task card with delete (trash) icon highlighted |
| `signout.png` | Header area showing Sign Out button |

## How to Take Screenshots

### macOS
- **Full screen:** `Cmd + Shift + 3`
- **Selection:** `Cmd + Shift + 4` (then drag to select area)
- **Window:** `Cmd + Shift + 4`, then press `Space`, then click window

### Windows
- **Snipping Tool:** `Win + Shift + S`

### Tips
- Use a clean browser (no extensions visible)
- Crop to show only the relevant part
- Keep file sizes reasonable (under 500KB each)
- Use PNG format for best quality

## After Adding Screenshots

The User Guide (`../USER_GUIDE.md`) will automatically display them!
